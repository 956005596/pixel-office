# 多 Agent 协作 UI

> 实时监控 Agent 状态、沟通记录和任务进度

## 🎯 功能特性

### 基础版 (index.html)
- ✅ **Agent 状态监控** - 显示每个 Agent 的闲忙状态
- ✅ **沟通记录** - 查看 Agent 之间的协作对话
- ✅ **任务看板** - 显示正在推进的任务
- ✅ **实时统计** - 活跃 Agent 数、进行中任务等
- ✅ **交互操作** - 刷新、新建任务、模拟消息

### 增强版 (workspace.html) ⭐
- ✅ **三栏布局** - Agent 列表 + 工作区 + 实时聊天
- ✅ **工作区互动** - 点击 Agent 查看详情、快速交互
- ✅ **实时同步** - 自动更新进度、模拟 Agent 消息
- ✅ **任务分配** - 可直接分配任务给空闲 Agent
- ✅ **快速操作** - 分配任务、查看进度、征求意见
- ✅ **实时状态** - 连接状态、时间显示
- ✅ **自动更新** - 每 5 秒模拟进度更新

### 实时版 (realtime.html) ⭐⭐⭐
- ✅ **真实 API 连接** - 连接 OpenClaw Gateway
- ✅ **WebSocket 实时推送** - Agent 状态实时更新
- ✅ **真实消息发送** - 可以真正和 Agent 交互
- ✅ **后端服务** - Node.js + Express + WebSocket

## 🚀 快速使用

### 方式 1：离线演示（不需要后端）

```bash
# 基础版 - 卡片式布局
open ~/.openclaw/workspace/projects/agent-collab-ui/index.html

# 增强版 - 三栏布局（推荐）
open ~/.openclaw/workspace/projects/agent-collab-ui/workspace.html
```

### 方式 2：实时版本（需要后端）⭐

```bash
# 1. 启动后端服务
cd ~/.openclaw/workspace/projects/agent-collab-ui/server
npm install  # 首次需要安装依赖
node server.js

# 2. 打开前端页面
open ~/.openclaw/workspace/projects/agent-collab-ui/realtime.html
```

**服务地址**：
- HTTP API: http://localhost:3100
- WebSocket: ws://localhost:3101
- OpenClaw Gateway: http://localhost:18789

## 🎮 交互功能

### 点击 Agent
- 查看 Agent 详细信息
- 查看当前任务和进度
- 发送消息交互

### 快速操作
- **分配任务** - 给 Agent 分配新任务
- **查看进度** - 请求进度汇报
- **征求意见** - 征求 Agent 建议

### 实时功能
- 每 5 秒自动更新任务进度
- 随机 Agent 发送消息
- 实时连接状态显示

## 💾 数据持久化

### 本地存储（localStorage）
- 聊天记录自动保存到浏览器本地存储
- 最近 100 条消息保留
- 页面刷新不会丢失数据
- 用户 ID 持久化

### 后端存储
- 数据存储位置：`~/.openclaw/workspace/projects/agent-collab-ui/server/data/`
- 支持最多 1000 条消息历史
- 多设备同步（同一浏览器）

### 功能按钮
- **📥 导出记录** - 导出所有聊天记录为 JSON 文件
- **🗑️ 清除记录** - 清除所有聊天记录（需确认）
