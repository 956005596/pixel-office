# AgentOS 角色形象设计文档

> 拟人化形象设计 - 专业亲切，辨识度高

---

## 设计原则

### 风格定位
- **参考风格**：《模拟人生》+《动物森友会》
- **特点**：卡通渲染但不过分夸张，有专业感又亲切
- **比例**：头身比约 1:3，略带 Q 版但不失专业

### 视觉统一规范
- **线条**：圆润流畅，无尖锐棱角
- **色彩**：与 UI 配色系统一致（Indigo/Violet 系）
- **表情**：温和友好，略带微笑
- **姿态**：站立或轻度侧身，自然放松

## 角色形象设计

### 1. 小松 (主 Agent / Orchestrator)

**定位**：协调者、团队核心

**形象描述**：
- **性别**：中性偏女性，短发利落
- **年龄**：25-28 岁
- **特征**：
  - 头戴松鼠耳朵造型的无线耳机（标志性）
  - 耳机为渐变紫色，带微弱发光效果
  - 短发，发色深棕带紫挑染
  - 穿简约白色衬衫 + 深色马甲
  - 手持平板设备（显示任务列表）
- **表情**：自信微笑，眼神专注
- **姿态**：站立，一手抱臂一手操作平板

**色彩**：白衬衫 + 深灰马甲 + 紫色耳机

**关键词**：活力、干练、亲和

---

### 2. 工程师 (Coder)

**定位**：技术实现、问题解决

**形象描述**：
- **性别**：男性
- **年龄**：26-30 岁
- **特征**：
  - 戴黑框眼镜，镜片有代码反射效果
  - 短发略凌乱，自然黑色
  - 穿深色连帽卫衣（带技术贴纸）
  - 牛仔裤，运动鞋
  - 背单肩包，露出键盘一角
  - 手腕上有智能手环
- **表情**：略带疲惫但专注的认真表情
- **姿态**：坐姿，双手在虚拟键盘上快速敲击

**色彩**：深灰卫衣 + 黑框眼镜 + 蓝色代码光效

**关键词**：极客、专注、可靠

---

### 3. 架构师 (Architect)

**定位**：系统设计、技术决策

**形象描述**：
- **性别**：男性
- **年龄**：32-38 岁
- **特征**：
  - 成熟稳重，留短胡须（山羊胡）
  - 戴细框金属眼镜
  - 短发整齐，略带银白
  - 穿高领毛衣 + 西装外套
  - 手持触控笔，面前悬浮蓝图/架构图
  - 戴简约腕表
- **表情**：沉稳思考，微微点头
- **姿态**：站立，一手托腮一手比划

**色彩**：深蓝毛衣 + 深灰外套 + 金色装饰

**关键词**：成熟、睿智、权威

---

### 4. 研究员 (Researcher)

**定位**：信息调研、数据分析

**形象描述**：
- **性别**：女性
- **年龄**：27-32 岁
- **特征**：
  - 知性优雅，长发扎成低马尾
  - 戴细边眼镜（可挂在胸前）
  - 穿浅色衬衫 + 针织开衫
  - 手持放大镜或数据平板
  - 身边漂浮数据图表和书籍图标
  - 戴简约耳环
- **表情**：好奇探索，眼睛发亮
- **姿态**：站立，一手拿资料一手记录

**色彩**：米色开衫 + 白衬衫 + 深棕长发

**关键词**：知性、敏锐、严谨

---

### 5. 编辑 (Editor)

**定位**：内容创作、文案撰写

**形象描述**：
- **性别**：中性偏女性
- **年龄**：24-28 岁
- **特征**：
  - 文艺气质，中长发微卷
  - 戴贝雷帽（深棕色）
  - 穿宽松毛衣 + 围巾
  - 手持羽毛笔或触控笔
  - 周围漂浮文字气泡和纸张
  - 戴多个细手链
- **表情**：灵感涌现，嘴角上扬
- **姿态**：坐姿，一手撑头一手书写

**色彩**：暖灰毛衣 + 棕色贝雷帽 + 橙棕围巾

**关键词**：文艺、灵感、细腻

---

### 6. 管家 (Assistant)

**定位**：日程管理、服务支持

**形象描述**：
- **性别**：男性
- **年龄**：30-35 岁
- **特征**：
  - 专业干练，短发整齐
  - 穿制服风格服装（西装马甲+领结）
  - 白色手套
  - 手持托盘或日程本
  - 戴单片眼镜（装饰性）
  - 领结为品牌色（紫罗兰）
- **表情**：礼貌微笑，专注倾听
- **姿态**：微微鞠躬，一手背后一手递物

**色彩**：深灰制服 + 白衬衫 + 紫罗兰领结

**关键词**：专业、礼貌、可靠

## 头像/图标设计

### 简化版头像（用于列表）

当需要小尺寸显示时，使用以下简化元素：

| 角色 | 简化元素 |
|------|----------|
| 小松 | 松鼠耳机 + 短发 |
| 工程师 | 黑框眼镜 + 连帽衫 |
| 架构师 | 细框眼镜 + 胡须 |
| 研究员 | 马尾 + 细边眼镜 |
| 编辑 | 贝雷帽 + 卷发 |
| 管家 | 领结 + 单片眼镜 |

### 状态指示

在头像角落叠加状态点：
- 🟢 在线：正常显示
- 🟡 忙碌：头像边缘轻微发光
- ⚪ 离线：头像变为灰度

## 技术实现建议

### 方案 A：SVG 矢量插画
- 使用 SVG 绘制角色
- 优点：可缩放、文件小、可动画
- 难度：需要设计资源

### 方案 B：Emoji 组合 + CSS
- 使用 Emoji + CSS 样式组合
- 优点：无需额外资源、跨平台兼容
- 难度：表现力有限

### 方案 C：图标字体/图标库
- 使用类似 Fluent Emoji 的开源资源
- 优点：统一风格、质量高
- 难度：可能需要自定义

### 推荐
当前使用 **方案 B** (Emoji)，后期可考虑 **方案 A** (SVG) 提升品质。
